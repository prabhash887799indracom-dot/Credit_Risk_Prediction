# Credit Risk Prediction Using Machine Learning

## 📌 Project Overview

This project predicts whether a loan applicant is a **Low Credit Risk** or **High Credit Risk** customer using Machine Learning algorithms. It helps financial institutions make better lending decisions and reduce loan defaults.

---

## 🎯 Objective

- Predict credit risk of loan applicants.
- Reduce loan defaults.
- Improve loan approval decisions.
- Compare the performance of different Machine Learning models.

---

## 📂 Dataset

Dataset Used: German Credit Dataset

Features:
- Age
- Sex
- Job
- Housing
- Saving Accounts
- Checking Account
- Credit Amount
- Duration
- Purpose

Target:
- Low Risk
- High Risk
